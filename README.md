# Windows Auto Backup to Google Drive

A simple and effective Python utility for automatically backing up files. It allows you to save copies both to local drives and to Google Drive cloud storage.

## ✨ Features
- **GUI Settings:** A user-friendly interface for selecting files and destination folders.
- **Flexibility:** Support for local copying (USB, HDD) and the Google Drive API.
- **Automation:** Script `runner.py` designed to run at system startup.
- **Login:** All operations are recorded in a file `backup_log.txt` for error checking.
